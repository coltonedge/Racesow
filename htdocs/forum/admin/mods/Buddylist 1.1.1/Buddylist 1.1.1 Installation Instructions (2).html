Buddylist 1.1.1
Introduction
I trust that you have installed the rest of the Buddylist already. If not, do so now please, by reading the text file.
This Buddylist version has been designed so it can be visible on any page of your forum. But of course, without a guide you wouldn't know where. That's why I have made this guide. First of all, I will discuss a few specific pages with detailed instructions. After that, I will give comprehensive information on how to add it to other pages than the ones below. You are in no way obligated adding them to different pages; all I want to do, is give an example to answer two frequently asked questions regarding the Buddylist.

This document describes the alternative way of listing the buddies, simply by comma-separating them and showing the online buddies in bold. Read this file if you want the original buddylist. You can also combine the two of course ;-).
Note: you will see a part of code called get_buddies_list($userdata);. This part retrieves the Buddylist from database, and links them to their profile. However, I have added an option so it automatically links them to sending a new private message. For doing so, use get_buddies_list($userdata, 'pm'); instead.
You can use this guide to install the Buddylist in the following pages:
  • View profile
  • Private messages section
  • Other...
In view profile
The Buddylist will be spread over the whole width of the page, underneath the actual profile. The buddies that will be shown, are the buddies from the user you are watching the profile, not your own buddies!
