Introduction
I trust that you have installed the rest of the Buddylist already. If not, do so now please, by reading the text file.
This Buddylist version has been designed so it can be visible on any page of your forum. But of course, without a guide you wouldn't know where. That's why I have made this guide. First of all, I will discuss a few specific pages with detailed instructions. After that, I will give comprehensive information on how to add it to other pages than the ones below. You are in no way obligated adding them to different pages; all I want to do, is give an example to answer two frequently asked questions regarding the Buddylist.

There is also another way of listing the buddies, simply by comma-separating them and showing the online buddies in bold. Read this file if you want that option instead of the list described in this document. You can also combine the two of course ;-).
You can use this guide to install the buddylist in the following pages:
  • View profile
  • Private messages section
  • Other...
In view profile

Other...
There are a few basic lines of code that you need to have the buddylist working. They are listed here.

For the template, you will have to search yourself where exactly you have/want to put the buddylist code. Note that you can also embed the code inside a table, alter the code and so on.
#
#-----[ OPEN ]--------------------------------------------
#
*.php

#
#-----[ FIND ]--------------------------------------------
#
//
// End session management
//

#
#-----[ BEFORE, ADD ]-------------------------------------
#
//
// Buddylist actions
//
$buddy_id = ( isset($HTTP_GET_VARS['b']) ) ? intval($HTTP_GET_VARS['b']) : 0;
$buddy_action = ( isset($HTTP_GET_VARS['buddy']) ) ? $HTTP_GET_VARS['buddy'] : '';
if( $buddy_id && $buddy_action != '' )
{
  if( $buddy_action == 'add' )
  {
    add_buddy($userdata['user_id'], $buddy_id);
  }
  else if( $buddy_action == 'remove' )
  {
    remove_buddy($userdata['user_id'], $buddy_id);
  }
}
//
// END: Buddylist actions
//

#
#-----[ FIND ]--------------------------------------------
#
$template->pparse('body');

include($phpbb_root_path . 'includes/page_tail.'.$phpEx);

?>

#
#-----[ BEFORE, ADD ]-------------------------------------
#
get_buddies($userdata);

#
#-----[ OPEN ]--------------------------------------------
#
# open the .tpl file that belongs to the PHP file you just edited
# make sure to edit this file for every template installed
#
templates/subSilver/*.tpl

#
#-----[ ADD ]---------------------------------------------
#
<table width="100%" cellspacing="0" cellpadding="0" border="0">

  </tr>
</table>
